Исправь мобильную версию: $ARGUMENTS

## КРИТИЧЕСКОЕ ПРАВИЛО
Ты меняешь ТОЛЬКО мобильную версию (ширина < 768px). Десктопная версия должна остаться ТОЧНО такой же как была.

## Как это сделать
1. Прочитай правило в .claude/rules/mobile-responsive.md
2. Найди компоненты которые нужно исправить
3. Для КАЖДОГО стиля который меняешь:
   - Сохрани текущее значение как `md:` версию (если ещё нет)
   - Поставь новое значение как базовый стиль (без префикса)
   
   Пример: было `className="text-5xl p-10"`
   Стало: `className="text-2xl md:text-5xl p-4 md:p-10"`

4. После изменений проверь:
   - DevTools → 375px ширина → мобилка выглядит хорошо
   - Полная ширина → десктоп НЕ изменился

## Типичные проблемы мобильной версии VoiceMap
- Огромные иконки → уменьши базовый размер, сохрани md: для десктопа
- Текст вылезает за экран → уменьши font-size, добавь truncate или break-words
- Карточки слишком широкие → grid-cols-1 на мобилке, md:grid-cols-12 на десктопе
- Отступы огромные → p-4 на мобилке, md:p-10 на десктопе
- Элементы в ряд не помещаются → flex-col на мобилке, md:flex-row на десктопе
